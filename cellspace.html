About

CellSpace is a HTML5/WebGL space shooter game built on cellular automata. All rule lookups and most of the collission detections are calculated on the GPU :)

Controls

  • mouse click to shoot
  • cursor keys for navigation
  • 'b' to fire many shots at once
  • 'space' to reset the game
This is is an early beta realease please direct bug reports or ideas to the EvoCell github page
